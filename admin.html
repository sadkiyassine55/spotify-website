Database setup steps

  1. Create MySQL/PostgreSQL database: spotifyhub_db
  2. Run these tables: 
    -- keys table
CREATE TABLE keys (
  id INT AUTO_INCREMENT PRIMARY KEY,
  key_code VARCHAR(50) UNIQUE,
  status ENUM('UNUSED','BINDING','REUSABLE') DEFAULT 'UNUSED',
  bound_email VARCHAR(255),
  invite_link TEXT,
  created_at DATETIME,
  redeemed_at DATETIME
);

-- invite_links table
CREATE TABLE invite_links (
  id INT AUTO_INCREMENT PRIMARY KEY,
  link TEXT,
  status ENUM('AVAILABLE','ASSIGNED') DEFAULT 'AVAILABLE',
  assigned_key VARCHAR(50),
  assigned_to_email VARCHAR(255)
);

-- activity_log (optional)
CREATE TABLE activity_log (
  id INT AUTO_INCREMENT PRIMARY KEY,
  action VARCHAR(100),
  details TEXT,
  created_at DATETIME
);
  3. Update backend (PHP/Node) to connect with credentials.
  4. Replace simulation functions with actual DB queries.
  5. Set up session authentication for admin (only you).

Use "spotifygold" as master password example.
